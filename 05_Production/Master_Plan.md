# 📋 Master Plan — Spark of Defiance

> **Цель:** Полная игра (3 зоны + 3 босса + хаб + финал)  
> **Срок:** Не определён — качество важнее скорости  
> **Формат:** Портфолио для вуза + инди-релиз  
> **Целевой FPS:** 60  
> **Платформа:** PC (Windows)

---

## 🗺️ Фазы разработки

### Фаза 1: Фундамент (CP-0 — CP-3)
**Цель:** Работающий прототип с движением, стелсом и замедлением времени.

| CP | Название | Содержимое | Зависимости |
|----|----------|------------|-------------|
| **CP-0** | Project Setup | Инициализация Godot, viewport, input map, структура папок, AutoLoad каркасы | — |
| **CP-1** | Player Movement | Бег, прыжок, wall-slide, wall-jump, dash, roll, crouch, ledge grab | CP-0 |
| **CP-2** | Stealth System | Конус зрения, шум, тревога, состояния врага, укрытия | CP-1 |
| **CP-3** | Time Dagger | Замедление мира, шейдер, энергия, TimeManager | CP-2 |

## 🎯 Критерии приёмки CP (для Романа)

Каждый CP считается завершённым, когда:
1. Builder сдал код.
2. Architect провёл ревью (KP = 0).
3. Роман протестировал по чеклисту ТЗ и подтвердил работоспособность.
4. Код влит в Git (commit с тегом `cp-N`).

## 📊 Риски и митигация

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Wall-run баги | Средняя | Высокое | Тестировать на TileMap с разными углами с самого CP-1 |
| Time Dagger десинхронизация | Средняя | Высокое | Не использовать `Engine.time_scale`, кастомная группа |
| Стелс производительность | Низкая | Среднее | `PhysicsDirectSpaceState2D` вместо 100 RayCast2D |
| Сохранение коррупция | Низкая | Высокое | Версионирование JSON, бэкапы, валидация |
| Арт не готов к CP-13 | Высокая | Среднее | Placeholder'ы на всех ранних CP, Leonardo для концептов |
| Scope creep (расползание) | Средняя | Высокое | Жёсткий Master Plan, новые фичи только через Architect |
