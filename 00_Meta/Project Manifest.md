## V. ФИЗИКА И ДВИЖЕНИЕ

| ID | Вопрос | Ответ | Архитектурное влияние |
|----|--------|-------|----------------------|
| V1 | Бег по стенам | **Вертикальный wall-slide (автоматическое замедление при касании)** | `is_on_wall()` + `velocity.y *= wall_slide_friction` |
| V2 | Wall-run | **Автоматический при касании** | Не требует удержания кнопки, просто касание стены |
| V3 | Wall-jump | **Меняет направление (импульс в сторону и вверх)** | `velocity.x = wall_jump_push * -facing`, `velocity.y = wall_jump_force` |
| V4 | Рывок (Dash) | **4 направления. В воздухе 1 раз (возвращается при касании земли)** | `dash_direction` из ввода, `can_air_dash` флаг, сброс на `is_on_floor()` |
| V5 | i-frames переката | **0.4 секунды** | `Timer` на 0.4s, флаг `is_invincible` |
| V6 | Приседание | **Влияет на всё: скорость ниже, шум тише, хитбокс меньше** | `CollisionShape2D` смена формы, `noise_radius` уменьшается, `speed` *= 0.5 |
| V7 | Двойной прыжок | **Открывает Мастер Джаан (Air Dash / Рывок в воздухе)** | Изначально недоступен, флаг `unlocked_double_jump` в ProgressionManager |
| V8 | Крюк-кошка | **Точка-точка, можно повиснуть и раскачаться (физика маятника)** | `RayCast2D` для прицеливания, `PinJoint2D` или кастомная физика маятника |
| V9 | Скорость | **Фиксированная с плавным ускорением/торможением** | `acceleration` и `friction` в `_physics_process`, не мгновенная смена скорости |
| V10 | Скольжение | **Замедляет падение** | `velocity.y = min(velocity.y, wall_slide_max_speed)` |
| V11 | Ledge grab | **Да, можно зацепиться и подтянуться** | `RayCast2D` вверх от рук, проверка уступа, анимация подтягивания |

## G. ГЕЙМПЛЕЙ И СТЕЛС

| ID | Вопрос | Ответ | Архитектурное влияние |
|----|--------|-------|----------------------|
| G1 | Обнаружение | **Градиентное (Idle -> Suspicious -> Alert)** | State Machine для врагов: `IdleState`, `SuspiciousState`, `AlertState` |
| G2 | Здоровье Кая | **HP (3 сердца, аптечки в мире)** | `health: int = 3`, `max_health = 3`, аптечки — подбираемые `Area2D` |
| G3 | Враги после тревоги | **Возвращаются через 30 сек, если потеряли Кая** | `Timer` 30s в `AlertState`, переход в `ReturnToPatrolState` |
| G4 | Обнаружение трупов | **Да** | Враги проверяют `Area2D` рядом с телами, переход в `SuspiciousState` |
| G5 | Скрытые убийства | **Просто кнопка** | Не QTE. Проверка расстояния и угла (со спины) |
| G6 | Снайперка | **Нажатие кнопки в направлении курсора** | `RayCast2D` от Кая в направлении `aim_direction` (стик/мышь) |
| G7 | Управление снайперкой | **Стик (геймпад)** | Правый стик для прицеливания, ЛКМ/кнопка для выстрела |
| G8 | Боезапас | **Ограниченный, подбираемый на уровнях** | `ammo: int`, `max_ammo = 5`, патроны — подбираемые `Area2D` |
| G9 | Макс патронов | **5** | `const MAX_AMMO: int = 5` |
| G10 | Паралич | **Можно убить ножом** | Состояние `paralyzed`, таймер 10s, доступно stealth_kill |
| G11 | Нож | **Только со спины (или сбоку, если враг полностью оглушен)** | Проверка угла: `abs(angle_to_player) > 120°` для спины |
| G12 | Лобовая атака ножом | **Кай получает 1 урон, отбрасывается, враг бьет тревогу** | `take_damage(1)`, `knockback()`, enemy переходит в `AlertState` |
| G13 | Дроны | **Летающие, маршруты заданные (по точкам)** | `Path2D` + `PathFollow2D`, точки маршрута |
| G14 | Страж SUN | **Движется по наземному маршруту** | `Path2D` на земле или `CharacterBody2D` с патрульной логикой |
| G15 | Кремниевая Тварь | **Реагирует на любой звук** | `Area2D` слуха с большим радиусом, не зрение |
| G16 | Элитный снайпер | **Статичный на вышке** | `StaticBody2D` или `Marker2D`, поворот только для прицеливания |
| G17 | Приманки | **Камень (бесконечный). 1 активная приманка** | `decoy_stone.tscn`, бросок через `apply_impulse`, шум при падении |
| G18 | Дым | **Блокирует линию видимости (конусы обрываются о дым)** | `Area2D` с `CollisionLayer` для визуальной блокировки RayCast |
| G19 | ЭМИ | **Падают на землю и выключаются на 10 сек** | `velocity.y = 0`, `gravity_scale = 1`, таймер 10s, затем `gravity_scale = 0` и возврат на маршрут |
| G20 | Тревога | **Спавн 2 новых врагов из невидимых зон** | `SpawnPoint` ноды, `instantiate()` врага при тревоге |
| G21 | Укрытия | **Только в приседе** | Высокая трава — `Area2D`, проверка `is_crouching` |
| G22 | Тени | **Динамические (от фонарей через LightOccluder2D)** | `PointLight2D` + `LightOccluder2D` на стенах/объектах |
