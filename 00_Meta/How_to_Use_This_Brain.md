# How to Use This Brain

## For Obsidian
1. Open folder as Vault
2. Enable Settings → Files and links → Use [[Wikilinks]]
3. Press Ctrl+G for graph view
4. Start from [[Index]] or [[00_Meta/Start_Here]]

## For New Chat with Kimi (Architect)
1. Open [[00_Meta/New_Chat_Onboarding]]
2. Copy entire contents
3. Paste into new chat as first message
4. Add current CP context

## For Builder (DeepSeek)
1. Kimi issues TZ in CP-N format
2. Copy TZ to Builder chat
3. Builder returns code
4. Show code to Kimi for review
5. Kimi gives KP/NC → Builder fixes

## For Leonardo.Ai
1. Kimi writes prompt in [[99_Templates/Asset_Request_Template]]
2. Copy prompt to Leonardo
3. Generate, download PNG
4. Place in `assets/art/leonardo/`
5. Report to Kimi → updates [[04_Assets/Asset_Inventory]]

## For Godot
1. Open `06_Project/project.godot` (when created)
2. Work in `06_Project/`
3. Do not edit `.godot/` or `*.tmp`
4. Commit via Git

## Daily Workflow
1. Open [[99_Templates/Daily_Notes_Template]] — create today's note
2. Check [[05_Production/CP_Index]] for current CP
3. Review [[00_Meta/Risk_Register]] before risky changes
4. Log decisions in [[00_Meta/Decision_Log]]
