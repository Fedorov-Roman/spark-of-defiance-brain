# Project Manifest v2

## Identity
- **Name:** Spark of Defiance
- **Universe:** Kill a Horse of Freedom (4717 AD, Postulate planet)
- **Format:** Portfolio (university) + Personal indie release
- **Scope:** Full game (3 Zones + 3 Bosses + Hub + Final)
- **Timeline:** Unrushed; quality over speed

## Tech
- **Engine:** Godot 4.6.3 stable
- **Renderer:** Forward+ (2D)
- **Physics:** Built-in 2D (no Jolt)
- **Script:** GDScript (strict typing)
- **VCS:** Git (GitHub)

## Platform
- **Primary:** PC (Windows)
- **Input:** Gamepad (XInput) + Keyboard/Mouse
- **Window:** Adaptive (base 640×360, stretch canvas_items, NEAREST)
- **Target FPS:** 60
- **Min Spec:** Any modern PC

## Art
- **Tile size:** 32×32
- **Viewport:** 640×360 (20×11 tiles)
- **Scale:** x2 (1280×720) or x3 (1920×1080)
- **Style:** Pixel-art, dark sci-fi, post-apocalyptic
- **Pipeline:** Placeholder (colored rects) → Leonardo concepts → final pixel-art

## Audio
- **Music:** Dynamic (ambient/combat states)
- **SFX:** Freesound / itch.io
- **Voices:** Text-only with placeholder blips
- **Language:** English
