## 🎯 Ключевые решения (Decision Log)

См. полный лог в [[05_Production/Decision Log\|Decision Log]].

- **Движок:** Godot 4.6.3 stable (PC Windows)
- **Жанр:** 2D Stealth-Platformer Metroidvania
- **Визуал:** Pixel-art 32×32, pixel-perfect x2/x3
- **Управление:** Геймпад + Клавиатура/Мышь
- **Стелс:** Градиентный (Idle → Suspicious → Alert)
- **Время:** Кастомный time_scale (мир 30%, Кай 100%)
- **Сохранение:** JSON, полное состояние мира
- **Хаб:** Одна динамическая сцена
- **Язык:** Английский
- **Leonardo:** Free Plan, 150 токенов/день

1. **Builder не проектирует.** Любое архитектурное изменение — через Kimi.
2. **Leonardo не знает контекста.** Все промпты — визуально самодостаточные (без имён «Кай», «SUN», «Постулат»).
3. **Резерв (Flash/Qwen) — только по запросу Kimi.** Не обращаться напрямую.
4. **Каждый CP начинается с ТЗ и заканчивается ревью.** Нет пропусков.
