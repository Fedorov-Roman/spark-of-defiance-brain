# 🏃 Movement & Acrobatics

> **Godot класс:** `CharacterBody2D` + `AnimatedSprite2D` / `AnimationTree`  
> **Физика:** `_physics_process(delta)`  
> **Версия:** Godot 4.6.3

## 1. Состояния движения (State Machine)

```gdscript
enum State {
    IDLE,
    RUN,
    JUMP,
    FALL,
    WALL_SLIDE,
    WALL_JUMP,
    DASH,
    ROLL,
    CROUCH,
    LEDGE_GRAB,
    HOOK_SWING
}
```

**Правило:** Только одно состояние активно. Переходы через `transition_to(new_state)`.

## 2. Wall Slide (Скольжение по стене)

- **Условие:** Кай касается стены (`is_on_wall()`), нажата кнопка направления к стене, падает.
- **Автоматический:** При касании стены в воздухе — автоматически.
- **Эффект:** Гравитация уменьшается (скольжение медленное), анимация wall-slide.
- **Отскок (Wall Jump):** Нажатие прыжка → импульс в сторону от стены и вверх. Меняет направление.
- **Godot:** `get_wall_normal()` для определения направления отскока.

## 3. Dash (Рывок)

- **Направления:** 4 (влево, вправо, вверх, вниз). НЕ диагонали.
- **В воздухе:** Доступен 1 раз. Возвращается при касании земли (`is_on_floor()`).
- **Двойной рывок:** Открывает Мастер Клинков Джаан (второй рывок в воздухе).
- **i-frames:** Нет (не путать с перекатом). Рывок — уклонение через расстояние.
- **Длительность:** ~0.2 сек, фиксированная скорость.
- **Godot:** `move_and_slide()` с импульсом, `Timer` для восстановления контроля.

## 4. Roll (Перекат)

- **i-frames:** 0.4 секунды.
- **Условие:** Нажатие кнопки при беге/приседании.
- **Эффект:** Хитбокс уменьшается, неуязвимость, анимация переката.
- **Godot:** `AnimationPlayer` с треком `CollisionShape2D.disabled` + `Area2D` для хитбокса.

## 5. Crouch (Приседание)

- **Скорость:** Снижена (50% от бега).
- **Шум:** Тише (радиус шума меньше).
- **Хитбокс:** Уменьшен (можно пролезть под препятствия).
- **Укрытие:** Высокая трава скрывает Кая только в приседе.
- **Godot:** Отдельный `CollisionShape2D` для приседания, переключение через `shape`.

## 6. Ledge Grab (Зацепление за уступ)

- **Условие:** Кай падает, руки на уровне уступа, нажата кнопка вверх.
- **Эффект:** Кай цепляется, анимация виса. Подтягивание — нажатие прыжка.
- **Godot:** `RayCast2D` вперёд и вниз для определения уступа. `Area2D` для триггера.

## 7. Hook / Grapple (Крюк-кошка)

- **Механика:** Точка-точка. Кай стреляет крюком в точку (курсор/стик), цепляется.
- **Физика:** Маятник. Можно раскачиваться и прыгать с крюка.
- **Визуал:** Пунктирная линия прицеливания.
- **Godot:** `RayCast2D` или `ShapeCast2D` для определения точки цепления. `PinJoint2D` или ручная физика (velocity к точке). `Line2D` для визуализации цепи.
- **Открытие:** Инженер Уно улучшает крюк (дальность, скорость).

## 8. Ускорение и торможение

- **Скорость:** Фиксированная максимальная, но с плавным ускорением и трением ("сочность").
- **Godot:** `velocity.x = move_toward(velocity.x, target_speed, acceleration * delta)`.
- **Трение:** При отпускании кнопки — замедление до 0 через `friction`.

## 9. Jump (Прыжок)

- **Высота:** Фиксированная (не зависит от удержания кнопки).
- **Coyote Time:** ~0.1 сек после отрыва от земли — можно прыгнуть.
- **Jump Buffer:** ~0.1 сек до касания земли — прыжок выполнится при приземлении.
- **Wall Jump:** Отскок от стены с импульсом в сторону и вверх.
