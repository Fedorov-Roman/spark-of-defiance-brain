# 🚀 Fast Travel — Истребитель «Вспышка»

## 1. Концепция

- **Модель:** Истребитель класса «Вспышка» (UFO).
- **Внешний вид:** Ярко-красный корпус, угольно-серые акценты, агрессивные обводы как у гоночного болида, складывающиеся крылья.
- **Свойства:** Высокая скорость, стелс-режим (маскировка от радаров SUN), превосходная манёвренность.

## 2. Механика

### Получение
- Кай находит заброшенные ангары UFO или угоняет истребители у SUN.
- Инженер Уно восстанавливает их в хабе после доставки нужных деталей.

### Использование
- На карте появляются посадочные платформы (7-10 штук на всю карту).
- Игрок вызывает «Вспышку» через терминал в хабе.
- **Автоматическая короткая кат-сцена** (не управляемый полёт).
- За считанные секунды перелёт между открытыми зонами.

### Godot реализация
```gdscript
# fast_travel_manager.gd (часть ProgressionManager или отдельный AutoLoad)

var unlocked_platforms: Array[String] = []
var current_platform: String = "hub"

func unlock_platform(platform_id: String) -> void:
    if platform_id not in unlocked_platforms:
        unlocked_platforms.append(platform_id)

func travel_to(platform_id: String) -> void:
    if platform_id not in unlocked_platforms:
        return
    # Сохранение состояния текущей зоны
    SaveManager.save_zone_state(current_platform)
    # Кат-сцена
    UIManager.show_travel_cutscene()
    await get_tree().create_timer(2.0).timeout
    # Смена сцены
    GameManager.change_scene("res://scenes/levels/" + platform_id + ".tscn")
    current_platform = platform_id
    # Загрузка состояния новой зоны
    SaveManager.load_zone_state(platform_id)
```

## 3. Кат-сцена

- **Длительность:** 2-3 секунды.
- **Визуал:** Взлёт «Вспышки», звёзды, переход через гиперпространство (эффект искажения), посадка.
- **Звук:** Рев двигателей, гиперпрыжок.
- **UI:** Название зоны назначения.

## 4. Посадочные площадки

- **Визуал:** Площадка с маркерами UFO, мигалками, топливные шланги.
- **Взаимодействие:** Подход + кнопка «E» → меню выбора направления.
- **Условие:** Нужно сначала найти площадку в зоне, чтобы она появилась в списке.

---

*«Вспышка» — связующее звено между зонами. Экономит время и добавляет масштаб.*
