# 📈 Progression & Hub System

## 1. Система прогрессии

Прогрессия жёстко привязана к спасению 6 NPC. **Нет прокачки за опыт.**

### NPC-мастера

| NPC | Где спасти | Что даёт | Визуальное изменение хаба |
|-----|------------|----------|---------------------------|
| **Уно** (Инженер) | Зона 1, клетка | Крюк-кошка, улучшения гаджетов, лавка с боеприпасами | Появляется мастерская, верстак, инструменты |
| **Лиара** (Медик) | Зона 2, ловушка | +1 сердце HP, аптечки в хабе | Появляется лазарет, мед-оборудование, зелёные тона |
| **Джаан** (Мастер Клинков) | Зона 2, арена | Двойной рывок в воздухе, +урон ножа | Тренировочный манекен, оружейная стойка |
| **Элия** (Хрономант) | Зона 3, скрытая комната | Кинжал Времени (полная сила), «Эхо» (невидимость после рывка) | Мистическая зона, синие кристаллы, артефакты |
| **Кесс** (Пилот) | Зона 1, разбитый истребитель | Fast Travel (Вспышка), +10% скорость бега | Ангар, части истребителя, карта на стене |
| **Финч** (Связист) | Зона 1/2, ретранслятор | Карта мира, маркеры секретов | Коммуникационная панель, экраны, антенны |

### Валюта: Дата-драйвы (Data Drives)

- **Назначение:** Покупка/активация улучшений в хабе.
- **Где найти:** Разбросаны по зонам (секретные места, зачищенные враги).
- **Визуал:** Маленькие флешки/кристаллы, светятся синим.

## 2. Хаб «Искра»

### Начальное состояние
- Тёмная пещера с парой партизан.
- Пустые места для мастерских (маркеры на полу).
- Базовый костёр, ящики, тенты.

### Динамическое изменение

Каждый спасённый NPC:
1. Появляется в хабе (visible = true).
2. Активирует свою зону (мастерская, лазарет, ангар).
3. Добавляет диалоги и функционал.

### Терминал «Вспышки»
- Появляется после спасения Кесс.
- Интерактивный объект для выбора зоны fast-travel.
- Требует найденные посадочные площадки.

## 3. Godot реализация

### ProgressionManager
```gdscript
var rescued_npcs: Dictionary = {
    "uno": false,
    "liara": false,
    "jaan": false,
    "elia": false,
    "kess": false,
    "finch": false
}

var data_drives: int = 0

func rescue_npc(npc_id: String) -> void:
    rescued_npcs[npc_id] = true
    npc_rescued.emit(npc_id)
```

### Hub Scene
```gdscript
# hub.gd
func _ready() -> void:
    _update_hub_state()

func _update_hub_state() -> void:
    $UnoWorkshop.visible = ProgressionManager.rescued_npcs.get("uno", false)
    $LiaraMedbay.visible = ProgressionManager.rescued_npcs.get("liara", false)
    # ... и так далее для всех NPC
```
