# GDD Core

## Game Identity

| Parameter | Value |
|-----------|-------|
| **Title** | Spark of Defiance |
| **Genre** | 2D Stealth Platformer with Metroidvania elements |
| **Engine** | Godot 4.6.3 stable |
| **Platform** | PC (Windows) |
| **Input** | Gamepad + Keyboard/Mouse |
| **Base Resolution** | 640×360 (pixel-perfect x2/x3) |
| **Tile Size** | 32×32 |
| **Target FPS** | 60 |
| **Language** | English |
| **Art Style** | Pixel-art (placeholder → final) |

## Scope

- 3 Zones + Hub
- 3 Bosses
- 6 NPCs (rescue → abilities)
- 4 Enemy types + Elite variant
- Fast travel system (Vspyshka fighter)

## Core Pillars

1. **Stealth over combat** — Lethal frontal attacks = death.
2. **Acrobatic freedom** — Wall-slide, dash, grapple, ledge grab.
3. **Time manipulation** — Signature ability, tactical use.
4. **Metroidvania progression** — Rescue NPCs → unlock abilities → backtrack.
5. **Atmospheric storytelling** — Silent protagonist, lore via collectibles.
