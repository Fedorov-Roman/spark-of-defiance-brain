# UI/UX System v2

## HUD (In-Game)
- **Hearts:** Top-left. 3 hearts. Half-heart support (if needed). Blink on damage.
- **Ammo:** Top-left below hearts. "5/5" with sniper icon.
- **Gadgets:** Top-right. Icons for EMP, Smoke, Decoy with counts.
- **Energy:** Bottom-center. Thin bar (Time Dagger). Blue fill.
- **Noise Indicator:** Bottom-center below energy. Ripple waves when moving.
- **Alert Indicator:** Near enemy. "!" icon + red flash.
- **Objective Marker:** Edge of screen arrow (if off-screen).

## Menus
- **Pause:** Resume, Options, Save & Quit, Restart from Checkpoint.
- **Options:** Graphics (resolution, fullscreen, vsync), Audio (master, music, sfx), Controls (rebind, gamepad/KB), Accessibility (text speed).
- **Map:** Full-screen. Discovered areas only. Fast travel from here (if Kess rescued).
- **Inventory:** Gadgets, Data Drives count, lore logs.

## Dialogue
- Text box: Bottom 1/4 of screen. Pixel portrait left. Name top-left of box.
- Typewriter effect (char-by-char). Skippable with button.
- No voice. Text blip sound.

## Tutorial (Zone 1)
- Contextual pop-ups: "Press A to jump", "Hold LT to aim", "Crouch in grass to hide".
- Disappear after first use. Can be disabled in options.

## Visual Conventions
- Vision cones: semi-transparent, color-coded.
- Noise ripples: expanding circles from Kai's feet (white in stealth, red if heard).
- Grapple line: dashed cyan.
- Sniper line: dashed red while aiming.
