# 🏜️ Zone 1: Red Dunes (Красные Дюни)

> **Тип:** Обучение + Экшн  
> **TileMap:** 640×360 (1 экран) × 10 секций = 6400×360  
> **Секретов:** 5 Дата-драйвов  
> **Босс:** Песчаный Левиафан  
> **Враги:** Дроны (5), Стражи (3), Кремневые Твари (2)  
> **Посадочная площадка:** 2 штуки

## Визуальная концепция

- Бескрайние красные песчаные дюни, закатное оранжевое небо.
- Два маленьких луны на горизонте.
- Ржавые металлические руины (остатки баз SUN).
- Песок анимированный: `GPUParticles2D` с низкой плотностью (пыль).
- Parallax: 3 слоя (дюни далёкие, дюни средние, передний план).

## Механика зоны: Пыльная буря

- **Случайное событие:** Каждые 60-120 секунд, длится 40-60 сек.
- **Эффект:**
  - Видимость падает для обеих сторон (конус зрения сокращается на 50%).
  - Шум шагов Кая маскируется (радиус шума = 0).
  - Враги переходят в SEARCH (не видят, но слышат).
- **Визуал:** `CanvasModulate` затемнение + `GPUParticles2D` (песчаные частицы, высокая плотность).
- **Звук:** Ревущий ветер (амбиент).

## Структура уровня (секции)

### Секция 1: "Кратер" (обучение)
- **Цель:** Базовое движение (бег, прыжок, wall-slide).
- **Элементы:** Низкие платформы, руины со стенами для wall-slide.
- **Враги:** 1 Дрон (простой патруль).
- **Секрет:** 1 Дата-драйв (за wall-slide стеной).
- **Обучение:** Подсказки на экране ("Нажмите Пробел для прыжка", "Удерживайте стену для скольжения").

### Секция 2: "Руины" (стелс-введение)
- **Цель:** Первый стелс (укрытия, обход врага).
- **Элементы:** Высокая трава (только в приседе), стены, тени.
- **Враги:** 2 Дрона + 1 Страж (перекрёстный патруль).
- **Секрет:** 1 Дата-драйв (вентиляционная шахта, требует крюка-кошки — но крюк ещё недоступен, вернуться позже).

### Секция 3: "Провал" (платформинг)
- **Цель:** Wall-jump, рывок через пропасти.
- **Элементы:** Глубокие провалы, узкие стены, шипы внизу (1 урон).
- **Враги:** 1 Дрон (летает над пропастью).
- **Секрет:** 1 Дата-драйв (на вершине wall-jump цепочки).

### Секция 4: "Оазис" (передышка)
- **Цель:** Чекпоинт, аптечка, первый NPC (пленник, даёт подсказку).
- **Элементы:** Маленький зелёный оазис (вода, пальмы), контраст с красным песком.
- **Враги:** Нет.
- **Секрет:** 1 Дата-драйв (под водой, требует приседания).

### Секция 5: "Конвой" (боевое введение)
- **Цель:** Снайперка, ЭМИ, паралич.
- **Элементы:** Конвой SUN, дроны, турель (отключается ЭМИ).
- **Враги:** 2 Дрона + 1 Страж + 1 Турель.
- **Секрет:** 1 Дата-драйв (в грузовом контейнере, открывается после отключения турели).

### Секция 6-9: "Глубь" (прогрессия)
- Смешанные механики: стелс, платформинг, бой.
- Кремневые Твари (2 штуки) — звуковые ловушки.
- Посадочные площадки (2 штуки).

### Секция 10: "Арена Левиафана" (босс)
- Отдельная арена (640×360).
- Чекпоинт перед входом.
- Босс: Песчаный Левиафан.

## TileMap слои

| Слой | Назначение | Коллизия |
|------|------------|----------|
| 0 | Background (декор) | Нет |
| 1 | Ground (песок, платформы) | Да |
| 2 | Walls (руины, стены) | Да |
| 3 | Details (кости, мусор) | Нет |
| 4 | Cover (трава, тени) | Нет (Area2D overlay) |
| 5 | Spikes (шипы) | Да (1 урон) |
| 6 | Foreground (передний план) | Нет |

## Godot сцена

```gdscript
# zone_1_red_dunes.gd
extends Node2D

@onready var tilemap: TileMap = $TileMap
@onready var sandstorm_timer: Timer = $SandstormTimer
@onready var sandstorm_particles: GPUParticles2D = $SandstormParticles
@onready var canvas_modulate: CanvasModulate = $CanvasModulate

var _is_sandstorm: bool = false

func _ready() -> void:
    _start_sandstorm_cycle()
    AudioManager.play_ambient("red_dunes")

func _start_sandstorm_cycle() -> void:
    while true:
        await get_tree().create_timer(randf_range(60.0, 120.0)).timeout
        _start_sandstorm()
        await get_tree().create_timer(randf_range(40.0, 60.0)).timeout
        _end_sandstorm()

func _start_sandstorm() -> void:
    _is_sandstorm = true
    sandstorm_particles.emitting = true
    canvas_modulate.color = Color(0.6, 0.4, 0.3, 1.0)  # Тёмно-оранжевый
    # Уменьшаем конус зрения всех врагов
    for enemy in get_tree().get_nodes_in_group("enemies"):
        if enemy.has_method("reduce_vision"):
            enemy.reduce_vision(0.5)

func _end_sandstorm() -> void:
    _is_sandstorm = false
    sandstorm_particles.emitting = false
    canvas_modulate.color = Color(1.0, 1.0, 1.0, 1.0)  # Норма
    for enemy in get_tree().get_nodes_in_group("enemies"):
        if enemy.has_method("restore_vision"):
            enemy.restore_vision()
```
