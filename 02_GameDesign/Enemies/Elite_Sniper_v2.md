# Elite Sniper v2

## Stats
- HP: 2
- Stationary on tower/ledge.
- Vision: 60° narrow cone, 200 px range (long).
- Fires every 3 sec if Kai in sight.

## Behavior
- Scans left-right (slow).
- If Kai spotted: tracking laser (1 sec lock-on) → fires (1 heart).
- Cannot move. Can be paralyzed by Kai's sniper (10 sec).
- Drops sniper ammo on death.

## Visual
- Humanoid in ghillie-like camo, long rifle, blue scope glow.
- Laser sight visible as thin red line.
